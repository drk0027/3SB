Instrucciones de uso

Toda la base de datos de este sistema de blog es un archivo json simple. Los cambios realizados localmente se han exportado en el campo a continuacion para que lo almacene en reemplazo al archivo entradas.json con su editor de textos favoritos. En el momento de hacerlo, se actualizaran las nuevas entradas.

Aunque no deberia haber problemas en caso de manejo correcto, lo recomendable es siempre guardar una copia de un archivo entradas.json por si accidentalmente se altera la estructura del archivo. Si ocurriese esto, se cargaria el archivo incorrecto y no habria forma de restaurarlo
